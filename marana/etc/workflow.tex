\subsubsection{{\bfseries\sffamily TODO} Orchestrate chords}
\label{sec:orgafa9622}
Figure out mixtures that suit the following dynamics: 

\begin{itemize}
\item \emph{ppp}: almost all air noise from the winds, w/muted strings
\item \emph{pp} : pizzicato muted strings, single voiced winds
\item \emph{p}  : pizz. muted strings, sfp winds/brass and single
\item \emph{mp} : pizz. strings no mutes, sfp brass and single voiced wwinds
\item \emph{f}  : "pizz rhythms" now sfp on winds/brass, bowed strings
\item \emph{ff} : following the same pattern as above, dropping back to \emph{mp}
model and rising to a full \emph{ff} tutti at the points in the phrase
where the melody and chords coalesce
\end{itemize}
